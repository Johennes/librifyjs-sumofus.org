# LibrifyJS: sumofus.org

*Mirrors: [GitHub] ~ [GitLab]*

A Firefox extension for replacing the JavaScript blocked by GNU LibreJS on sumofus.org
with free / libre code.

## Status

### What Works

- Petition signing form

### What Doesn't Work

- Probably anything else
